=== BOT DISCORD — DEPLOY EN RAILWAY ===

1. Crea un nuevo proyecto en Railway
2. Sube esta carpeta (arrastra o conecta GitHub)
3. Ve a Variables y añade:
   - DISCORD_BOT_TOKEN = (tu token del bot)
   - SESSION_SECRET    = (cualquier texto, ej: miSecreto123)
4. El bot arranca solo con: node index.mjs

No necesita npm install ni compilación.
Todo está ya listo para ejecutarse.
